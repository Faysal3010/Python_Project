# Unnderstand

### GUI and CLI
---

#### CLU more powerful tool

show current path : `pwd`

show the file, folder, etc : `ls`

make folder : `mkdir` 

example:
```
mkdir folder_name
```
move specific path : `cd`

example:
```
cd folder_name
```

move to previous one step back : `cd ..`


make file : `touch`

example:
```
touch test.py
```
Remove file : `rm test.py`

```
rm test.py
```

Remove folder force way :  `rm -rf folder_name`

```
rm -rf folder_name
```
